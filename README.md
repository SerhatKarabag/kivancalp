# kivancalp

Unity 6 (`6000.3.3f1`) card-match prototype.

## Run

1. Open the project in Unity `6000.3.3f1`.
2. Open `Assets/Scenes/Game.unity`.
3. Press Play.

## Source Layout

- `Assets/Scripts/App`
- `Assets/Scripts/Core`
- `Assets/Scripts/Gameplay`
- `Assets/Scripts/Infrastructure`
- `Assets/Scripts/UI`
