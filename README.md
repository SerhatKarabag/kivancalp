# kivancalp

Unity 6 (`6000.3.3f1`) card-match prototype.

## Run

1. Open the project in Unity `6000.3.3f1`.
2. Open `Assets/Scenes/SampleScene.unity`.
3. Press Play.

## Source Layout

- `Assets/Game/Core`
- `Assets/Game/Gameplay`
- `Assets/Game/Infrastructure`
- `Assets/Game/UI`
- `Assets/Game/Tests`
